### Прогнозування вартості нерухомості за допомогою радіально-базисної нейронної мережі

На початку виконання програма (individual_task.py) обробляє вхідний датасет нерухомості (real_estate.csv), виявляючи закономірності в значеннях. Після цього користувач вводить свої дані для створення нового обʼєкту (вік нерухомості, відстань до найближчої станції метро та кількість продуктових магазинів в пішій доступності). Нейромережа аналізує дані, передбачає вартість такої нерухомості та виводить результати у вигляді графіку.
